# ☀️ Solar Agents Hub

Your 3-in-1 solar engineering toolkit.

## What's Inside

| Tool | What It Does |
|------|-------------|
| **solar-hub.py** | Central dashboard for all tools |
| **pv-multi-agent/** | AI-powered PV system design & simulation |
| **EnergyAI MCP** | Solar intelligence via opencode |

## Quick Start

1. **Streamlit Cloud**: Deploy solar-hub.py on share.streamlit.io
2. **Local**: Run `streamlit run solar-hub.py`
3. **PV Multi-Agent**: Run `streamlit run pv-multi-agent/gui.py`

### Requirements
- Python 3.12+
- Packages: streamlit, pvlib, pandas, numpy, plotly
